# AskNewton Society-of-Mind (TypeScript)

Option B scaffold: Express + Zod + BullMQ + pgvector.

## Quick start
1) `cp .env.example .env` and set values
2) `npm i`
3) Start Postgres + Redis (docker-compose coming soon)
4) `npm run dev`

## Structure
- `src/index.ts` – HTTP gateway
- `src/orchestrator/router.ts` – Planner/Router
- `src/agents/*` – Specialized agents
- `src/schemas/*` – Zod data contracts
- `src/tools/*` – Tool adapters (plan catalog, provider dir, OCR stubs)
- `src/queue/*` – BullMQ setup
- `src/utils/*` – OpenAI wrapper, logging
